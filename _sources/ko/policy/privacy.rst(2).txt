개인정보처리방침
=====================================================

1. 사이트에서 제공하는 리소스를 사용하는 경우 개인정보 수집에 동의한것으로 간주합니다. 동의하지 않을경우, 사이트를 나가시면 됩미다.
    - API
    - 기타 사이트에서 제공하는 정보

2. 이 사이트에서는 서비스 유지 및 개선을 목적으로 이용자의 최소한의 정보만 수집합니다. 수집하는 이용자 정보는 다음과 같습니다.
    - IP
    - 유저 에이전트
    - 쿠키
    - 접속 국가
    - 단말기정보(OS, 화면사이즈, 디바이스 아이디, 폰기종, 단말기 모델명)
    - 방문일시
    - 서비스 이용기록
    - 부정 이용기록

3. 사이트에서 개인정보를 수집하는 방법은 다음과 같습니다. 
    - API 사용과정에서 발생하는 정보
    - 사이트 접속시 클라이언트에서 보내는 정보
    - 기타 사이트에서 제공하는 리소스를 사용하는 과정중에서 생성, 취득되는 정보

4. 이용자의 개인정보는 아래와 같은 부분에서 사용되고 있습니다.
    - 사이트 통계
    - 부정 이용조회
    - 사이트 유지 및 개선

5. 이용자의 정보는 아래와 같은 방법으로 처리됩니다.
    - 이용자 정보는 마지막 이용일로부터 최대 10년간 보관 후 파기합니다.
    - 이용자 정보는 복구할수 없는 전자 방식으로 파기합니다.
