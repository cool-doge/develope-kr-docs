KakaoTalk 오픈채팅방 목록 검색¶

이 API는 카카오톡 오픈채팅방 목록을 검색 할수 있는 API입니다.

요청 URL¶

https://api.develope.kr/search/room/list

프로토콜¶

HTTPS

HTTP 메서드¶

GET

파라미터¶

파라미터

타입

필수 여부

설명

query

String

Yes

검색할 내용 또는 방 이름(링크)

type

String

Yes

검색할 방 타입 (m = 오픈채팅방 p = 오픈프로필(1:1))

page

int

No

검색할 페이지 (기본 1, 최대 1000)

count

int

No

한번에 검색할 방 갯수 (기본 30, 최대 100)

요청 예시¶

curl "https://api.develope.kr/search/room/list?query=Messenger Bot&type=m&page=1&count=30"
    -H "Content-Type:application/json"
