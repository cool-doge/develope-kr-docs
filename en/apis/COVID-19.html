COVID-19 Virus World Infection Information API¶

This API is the world’s information API for the COVID-19 virus.

경고

This API is running on a test server, not on the main server. The test is still in progress and may be unstable.

Request URL¶

https://api.corona.develope.kr/data

Protocol¶

HTTPS

HTTP Method¶

GET

Parameter¶

Parameter

Type

Required

Description

mode

String

Yes

Specify data to import (infected, dead, cures)

mode parameter value Description¶

Value

Description

i

Infected

d

Dead

c

Cured

Request example¶

curl "https://api.corona.develope.kr/data?mode=i"
    -H "Content-Type:application/json"
