COVID-19 바이러스 세계 감염 정보 API¶

이 API는 COVID-19 바이러스에 대한 세계 정보 API입니다.

경고

이 API는 메인 서버가 아닌 테스트 서버에서 구동되고 있습니다. 아직 테스트가 진행중이며, 불안정 할 수 있습니다.

요청 URL¶

https://api.corona.develope.kr/data

프로토콜¶

HTTPS

HTTP 메서드¶

GET

파라미터¶

파라미터

Yes

가져올 데이터 지정(감염자, 사망자, 완치자)

mode 파라미터 값 설명¶

값

설명

i

감염자 (Infected)

d

사망자 (Dead)

c

완치자 (Cured)

요청 예시¶

curl "https://api.corona.develope.kr/data?mode=i"
    -H "Content-Type:application/json"
