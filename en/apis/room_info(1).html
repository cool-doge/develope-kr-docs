KakaoTalk OpenChat Info Search¶

This API is for Searching Kakaotalk OpenChat Room Information¶

더 보기

Looking for an OpenChat Room list API ? Read KakaoTalk OpenChat List Search API document.

Request URL¶

https://api.develope.kr/search/room

Protocol¶

HTTPS

HTTP Method¶

GET

Parameters¶

Parameters

Type

Required

Description

room

String

Yes

Room name or Chat Link

Request example¶

curl "https://api.develope.kr/search/room?room=MessengerBot"
    -H "Content-Type:application/json"
