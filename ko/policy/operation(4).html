운영 약관¶

  1. 이 문서에서는 앞으로의 운영에 대한 내용을 기재합니다.

    • 이용자 리소스 접근 제한 사항

    • 비 정상적인 사용에 대한 예시

    • 리소스 접근 제한에 대한 예시

  2. Develope.kr은 사용자가 정상적인 사용을 하지 않는다고 판단할 경우, 사용자의 리소스 접근을 제한할수 있습니다. 제한 되는 사항은 다음과 같습니다.

    • API

    • 기타 사이트에서 제공하는 정보

  3. 비 정상적인 사용에 대한 예시

    • DDOS(분산 서비스 거부 공격), 스팸 등과 같은 정상적인 서비스를 방해하는 행위

    • 과도한 리소스 접근 시도

    • SQL Injection, XSS 등과 같은 해킹, 공격 시도

    • 기타 비 정상적인 사용이라고 판단이 되는 행위

이곳에 적힌 예시들은 어디까지나 《예시》입니다. 추가적인 비 정상적인 사용이라고 판단되는 행위를 적발할 경우, 위 사항에 기재되어있지 아니하더라도 사용자의 리소스 접근을 제한할수 있습니다.

  1. 리소스 접근 제한에 대한 예시

    • 사이트 접근 차단

    • 파트너쉽이 맻어진 다른 사이트에서의 리소스 접근 제한

    • 기타 사용자가 사이트로부터 정보를 취득할수 있는 방법 차단

  2. 이 사이트에서의 제공되는 API의 정보의 근원지중 일부는 공식이 아님을 밝힙니다. 사이트에서 제공되는 리소스를 사용할경우, 이에 대해 숙지를 한것으로 판단합니다.

    • 비공식 : KakaoTalk 오픈채팅 정보 검색
